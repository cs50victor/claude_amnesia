## Installation

### Install uv

Install uv following the [official installation guide](https://docs.astral.sh/uv/getting-started/installation/).

### Run setup.py

```bash
uv run -q --refresh https://raw.githubusercontent.com/cs50victor/claude_amnesia/main/setup.py
```
